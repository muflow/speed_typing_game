# Speed Typing Game

Speed Typing Gam is a fun way to challenge your typing speed and accuracy. In this game, your goal is to type as many words as possible before the time runs out. At the end of the game, your score will be displayed, indicating the number of words you managed to type correctly.

![This is an alt text.](./public/img/stg_screenshot.png 'screenshot of the UI.')

## Features

Simple Interface: The game has a user-friendly interface that makes it easy to play.
Timer: A countdown timer is displayed to let you know how much time is remaining.
Word Display: Words appear on the screen for you to type.
Score Display: At the end of the game, your score (number of words typed) is shown.
React and JavaScript: Developed using React and JavaScript, ensuring a smooth and interactive experience.

## Installation

1. Clone the repository:

```
git clone https://github.com/muflow/speed_typing_game.git
```

2. Navigate into the project directory:

```
cd speed_typing_game
```

3. Install dependencies:

```
npm install
```

## Usage

1. Start the development server:

```
npm run dev
```

2. Open your browser and navigate to http://localhost:5173/ to view the application.

3. Hit the START BUTTON.

4. write as fast as you can!
